<h1>Bright Mind</h1>
<p>Bright Mind is a Flask-based academic hub for students and educators.
It centralizes syllabi, past papers, notes, ebooks, and videos in one clean interface.</p>

<h2>What it offers</h2>
<p>- Fast access to study resources by category<br>
- Ebook previews with cover images<br>
- Video lessons with title and description<br>
- Secure admin upload for new materials<br>
- Contribution page for community-shared content</p>

<h4>A compact learning portal for organized study material and easy content sharing.</h4>
